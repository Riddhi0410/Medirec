## 💊 Medicine Recommendation System

The Medicine Recommendation System is an AI-powered web application that suggests appropriate medicines based on symptoms entered by the user. The system helps users quickly identify possible medications and provides helpful information about treatments.
The goal of this project is to assist users in finding potential medicines for common symptoms, while encouraging consultation with healthcare professionals for proper medical advice.


## ⚙️ Installation & Setup  

## 1️⃣ Clone the Repository  

git clone https://github.com/Riddhi0410/Medirec  
cd Medirec 

## 🔹 Backend Setup
cd backend  
python -m venv venv  
venv\Scripts\activate  
pip install -r requirements.txt  
uvicorn server:app --reload    

Backend will run on:    http://localhost:8000

## 🔹 Frontend Setup
cd frontend  
npm install  
npm start
